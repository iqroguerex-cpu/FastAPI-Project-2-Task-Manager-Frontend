The interactive user interface for TaskFlow. This dashboard allows users to manage their tasks with a focus on speed, clarity, and "lazy-friendly" interactions.

---

## ✨ UI Highlights
- **Smart Progress Tracking:** Real-time metrics showing completion percentage and pending workload.
- **Lazy-Add Workflow:** Quick-entry system that auto-assigns IDs to save user time.
- **Dynamic Filtering:** Instant search bar and priority-based filtering.
- **Visual Rewards:** Strike-through animations and status balloons upon completion.
- **Optimized for Mobile:** Fully responsive layout for management on the go.
